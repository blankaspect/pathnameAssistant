PathnameAssistant

PathnameAssistant is a JavaFX tool that doesn't do much at all. File-system locations can be dragged (for example, from a file manager) and dropped onto the main window of the application. The filenames or pathnames of the accumulated locations can then be copied to the system clipboard as text.

This is a preliminary document; more information on this project will be along later.
